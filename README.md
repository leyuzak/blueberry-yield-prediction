# blueberry-yield-prediction
Accurately estimating blueberry yield is a significant challenge due to the dependency on diverse factors such as weather patterns and pollination efficiency. This project addresses this complexity through a structured Data Science workflow:  * **Feature Engineering:** The dataset includes critical variables like `Honeybee`, `Bumblebee`, `Andrena`, and `Osmia` densities, alongside climatic data like `MaxUpperTRange` and `Precipitation`. * **Modeling Approach:** I implemented the **HistGradientBoostingRegressor**, which is particularly effective at handling non-linear relationships and potential missing values in agricultural datasets. * **Validation:** To ensure the model performs reliably on unseen data, I utilized **K-Fold Cross-Validation ($K=5$)**, achieving a consistent RMSE across all folds, which validates the model's stability. * **Accessibility:** Beyond the code, the project is deployed as a **Streamlit** application, allowing non-technical users to input environmental parameters and receive instant yield predictions.
